Letterboxd Ratings

Fetches the Letterboxd community star rating for each movie (matched by TMDB / IMDb ID) and writes it to the fields below. Ratings are refreshed by the Update Letterboxd Ratings scheduled task.

Where to show the rating

Adds a line like "★ 4.1/5 on Letterboxd (108,132 ratings)" at the end of the movie description. Visible in every Jellyfin client. Recommended.
Placeholders: {rating} (0.5–5 stars) and {count} (number of ratings).
Shows in the ratings row next to the IMDb star. Only enable if you don't already use Rotten Tomatoes critic scores — it occupies the same badge.
Replaces the IMDb/TMDb star rating with the Letterboxd score converted to a 10-point scale. Leave off to keep IMDb stars.

Fetching behaviour

Minimum 500. Be polite to Letterboxd — the default of 1500 works well.
Movies checked within this window are skipped when the task runs.
Sent with each request. Change only if requests start being rejected.
